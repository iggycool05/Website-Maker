🌐 OpenBuilder
An Open Source Website & App Builder for Everyone

OpenBuilder is an open source platform designed to make building websites and applications simple, fast, and accessible for both non-coders and developers.

Whether you prefer drag-and-drop design or writing full custom code, OpenBuilder gives you the flexibility to do both in one seamless environment.

🚀 Vision

The goal of OpenBuilder is to bridge the gap between no-code and full-code development by creating a tool where:

Beginners can build without coding
Developers can fully customize everything
Everyone can collaborate in one ecosystem
✨ Features
🧩 For Non-Coders
Drag-and-drop interface
Pre-built components (buttons, text, layouts, etc.)
Visual styling tools (fonts, colors, spacing)
Real-time preview
💻 For Developers
Full access to HTML, CSS, and JavaScript
Live code editing
Modular file structure
Extensible plugin system
🔄 Hybrid Workflow
Switch between visual editor and code editor anytime
Edit generated code directly
Instant updates in preview
⚡ Core Capabilities
Live preview rendering
Component-based architecture
Custom styling system
Export full project files
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Architecture: Modular ES6 structure
Preview Engine: iframe-based live rendering
Future Plans:
React-based UI
Backend integration
Cloud saving
